Project Overview

My second project, titled "LKPD 2," focuses on creating a website to commemorate Hari Kesaktian Pancasila (Pancasila Sanctity Day). The website presents information about the history of the October 1st commemoration, dedicated to honoring Pancasila as the national ideology and the fallen revolutionary heroes of the G30S incident. It includes several key sections, such as a brief introduction to the commemoration, a table of contents with links to topics like history, the G30S incident, revolutionary heroes, and Lubang Buaya.

Additionally, there is an interactive table providing detailed information about the event, including the date, historical background, and the names of revolutionary heroes like Lieutenant General Anumerta Ahmad Yani and Major General R Suprapto. This project allowed me to further develop my HTML and CSS skills while gaining experience in organizing information with a clear structure and an appealing design.
